```bash

```
para os alunos que o git inicou na branch `master` use o comando abaixo:
```bash
git checkout -b main
```

apos executar o primeiro commit
``
git add .
git commit -m "primeiro commit"
```
para ver as alterações pendentes:
```
git status 
```
